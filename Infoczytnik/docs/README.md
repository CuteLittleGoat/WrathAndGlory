# Infoczytnik (DataSlate)

## 🇵🇱 Instrukcja dla użytkownika (PL)
Infoczytnik służy do wysyłania i odbierania klimatycznych komunikatów na żywo.

### Ekran menu
1. Otwórz `Infoczytnik/index.html`.
2. Wybierz, czy chcesz uruchomić wersję produkcyjną czy testową.

### Dla prowadzącego (GM)
1. Otwórz panel GM (`GM.html` lub `GM_test.html`).
2. Ustaw wygląd wiadomości: tło, logo, font, kolory, rozmiar tekstu.
3. Wpisz komunikat główny oraz opcjonalne prefix/suffix.
4. Kliknij **Wyślij**, aby natychmiast pokazać wiadomość graczom.
5. Użyj **Ping**, aby wysłać szybki sygnał dźwiękowy.
6. Użyj **Wyczyść komunikat**, aby usunąć treść z ekranu graczy.

### Dla graczy
1. Otwórz `Infoczytnik.html` lub `Infoczytnik_test.html`.
2. Po pierwszym wejściu kliknij ekran, aby odblokować audio w przeglądarce.
3. Ekran automatycznie aktualizuje się po każdej zmianie wysłanej przez GM.

## 🇬🇧 User instructions (EN)
Infoczytnik (DataSlate) is for live, thematic message broadcasting.

### Menu screen
1. Open `Infoczytnik/index.html`.
2. Choose production or test variant.

### For GM
1. Open GM panel (`GM.html` or `GM_test.html`).
2. Configure message visuals: background, logo, font, colors, text size.
3. Enter main message and optional prefix/suffix.
4. Click **Send** to broadcast instantly.
5. Use **Ping** for quick audio signal.
6. Use **Clear message** to remove player-side content.

### For players
1. Open `Infoczytnik.html` or `Infoczytnik_test.html`.
2. On first visit, click screen once to unlock browser audio.
3. View updates automatically whenever GM sends changes.
