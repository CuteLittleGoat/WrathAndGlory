# Prefixy i Suffixy — moduł Infoczytnik

Poniżej znajduje się kompletna lista wszystkich możliwych prefixów i suffixów używanych przez template/fakcje w module **Infoczytnik**, wraz z przypisanymi fontami.

Źródło danych: `Infoczytnik/GM_test.html` (stałe `FONT_STACK` i `LAYOUTS`) oraz import fontów w `Infoczytnik/Infoczytnik_test.html`.

## Fonty użyte globalnie (Google Fonts + fallback)

- Share Tech Mono
- Cinzel
- Rajdhani
- Black Ops One
- Staatliches
- Orbitron
- Questrial
- Russo One
- Fallback wspólny: `Calibri, Arial, sans-serif`
