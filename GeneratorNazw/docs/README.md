# Generator Nazw

## 🇵🇱 Instrukcja dla użytkownika (PL)
Ten moduł tworzy klimatyczne nazwy (postaci, statków, organizacji i innych elementów świata).

1. Otwórz `GeneratorNazw/index.html`.
2. Wybierz **Kategorię**.
3. Wybierz **Opcję** (typ nazwy w danej kategorii).
4. (Opcjonalnie) wpisz własny **seed** — dzięki temu otrzymasz powtarzalny wynik.
5. Ustaw **Ile** nazw wygenerować.
6. Kliknij **Generuj**.
7. Kliknij **Kopiuj wynik**, aby przenieść listę do schowka.
8. Przełącz język przez listę PL/EN.

## 🇬🇧 User instructions (EN)
This module creates thematic names (characters, ships, organizations, and more).

1. Open `GeneratorNazw/index.html`.
2. Choose a **Category**.
3. Choose an **Option** (name type inside that category).
4. (Optional) provide a **seed** for repeatable output.
5. Set how many names you want.
6. Click **Generate**.
7. Click **Copy result** to copy the full output.
8. Use the PL/EN language switch.
