# Audio — instrukcja użytkownika / User guide

Moduł **Audio** jest w przygotowaniu i na razie wyświetla stronę informacyjną „Strona w budowie”.

---

## Instrukcja obsługi (PL)

### Jak uruchomić
1. Otwórz stronę modułu: `Audio/index.html`.
2. Na ekranie zobaczysz komunikat „Strona w budowie” oraz krótką informację o przygotowaniu modułu.

### Co dalej
W kolejnych wersjach moduł będzie służył do odtwarzania sampli dźwiękowych.

---

## User guide (EN)

The **Audio** module is under construction and currently shows an informational “Strona w budowie” page.

### How to launch
1. Open the module page: `Audio/index.html`.
2. You will see the “Strona w budowie” message with a short note that the module is being prepared.

### What’s next
Future versions will provide audio sample playback.
