# Audio

## 🇵🇱 Instrukcja dla użytkownika (PL)
Audio służy do odtwarzania efektów dźwiękowych i zarządzania listami „Ulubione”.

### Widok użytkownika
1. Otwórz `Audio/index.html`.
2. Przejdź do listy dźwięków przygotowanej przez administratora.
3. Kliknij wybrany dźwięk, aby odtworzyć.
4. Przełączaj zakładki nawigacji, aby zmieniać widoki list.

### Widok administratora
1. Wejdź przez `Audio/index.html?admin=1`.
2. Kliknij **Wczytaj manifest**, aby pobrać aktualny katalog SFX.
3. Użyj wyszukiwarki i filtrów tagów, aby zawęzić listę.
4. Dodawaj elementy do list ulubionych.
5. Ustal, które listy mają być widoczne dla użytkowników jako główny widok.
6. Użyj **Odśwież ulubione**, aby pobrać bieżący stan zapisów.

## 🇬🇧 User instructions (EN)
Audio is used to play sound effects and manage favorites lists.

### User view
1. Open `Audio/index.html`.
2. Go to the sound list prepared by the admin.
3. Click a sound item to play it.
4. Use navigation tabs to switch between list views.

### Admin view
1. Open `Audio/index.html?admin=1`.
2. Click **Load manifest** to fetch current SFX catalog.
3. Use search and tag filters to narrow results.
4. Add sounds to favorites lists.
5. Decide which lists are visible as main user view.
6. Use **Refresh favorites** to sync current saved state.
