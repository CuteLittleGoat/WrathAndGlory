# Kalkulator

## 🇵🇱 Instrukcja dla użytkownika (PL)
Moduł ma dwa główne narzędzia: kalkulator kosztów PD oraz arkusz tworzenia postaci.

### Strona startowa modułu
1. Otwórz `Kalkulator/index.html`.
2. Wybierz jeden z przycisków:
   - **Kalkulator PD**,
   - **Tworzenie Postaci**.

### Kalkulator PD
1. Otwórz `Kalkulator/KalkulatorXP.html`.
2. Dla atrybutów i umiejętności wpisz wartość aktualną oraz docelową.
3. Suma kosztu aktualizuje się automatycznie.
4. Użyj **Resetuj wartości**, aby wyzerować pola.
5. W tabeli pod spodem znajdziesz maksymalne wartości atrybutów dla ras.

### Tworzenie Postaci
1. Otwórz `Kalkulator/TworzeniePostaci.html`.
2. Ustaw pulę PD.
3. Uzupełnij atrybuty (w tym Szybkość), umiejętności oraz wydatki dodatkowe.
4. Obserwuj komunikaty o błędach (przekroczenie puli, zasada „Tree of Learning”).
5. Przycisk **Instrukcja/Manual** otwiera PDF pomocy.
6. Przycisk **Maksymalne wartości atrybutów** otwiera tabelę limitów ras.

## 🇬🇧 User instructions (EN)
This module has two core tools: XP/PD cost calculator and character creation sheet.

### Module home
1. Open `Kalkulator/index.html`.
2. Choose:
   - **Kalkulator PD** (XP calculator),
   - **Tworzenie Postaci** (character creation).

### XP/PD calculator
1. Open `Kalkulator/KalkulatorXP.html`.
2. Enter current and target values for attributes and skills.
3. Total cost updates automatically.
4. Use **Reset values** to clear inputs.
5. The lower table shows maximum attribute values by species.

### Character creation
1. Open `Kalkulator/TworzeniePostaci.html`.
2. Set XP/PD pool.
3. Fill attributes (including Speed), skills, and extra expenses.
4. Watch validation messages (pool overflow, “Tree of Learning” rule).
5. **Instrukcja/Manual** opens a language-specific PDF guide.
6. **Maximum attribute values** opens species limits table.
