# DiceRoller

## 🇵🇱 Instrukcja dla użytkownika (PL)
DiceRoller symuluje rzuty kośćmi do testów Wrath & Glory.

1. Otwórz `DiceRoller/index.html`.
2. Ustaw:
   - **Poziom trudności**,
   - **Pula kości**,
   - **Liczba kości Gniewu**.
3. Kliknij **Rzuć**.
4. Zobaczysz animację i podsumowanie wyniku (sukces/porażka, efekty specjalne).
5. Przełącz język listą w prawym górnym rogu.
6. Kliknij **Strona Główna**, aby wrócić do launchera.

## 🇬🇧 User instructions (EN)
DiceRoller simulates dice tests for Wrath & Glory.

1. Open `DiceRoller/index.html`.
2. Set:
   - **Difficulty**,
   - **Dice pool**,
   - **Wrath dice count**.
3. Click **Roll**.
4. You get animation and a result summary (success/failure, special outcomes).
5. Use the top-right language switch.
6. Click **Main Page** to return to launcher.
