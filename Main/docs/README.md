# Main — Kozi Przybornik

## 🇵🇱 Instrukcja dla użytkownika (PL)
Main to ekran startowy, z którego uruchamiasz pozostałe moduły.

### Jak wejść i co zobaczysz
1. Otwórz `Main/index.html`.
2. Zobaczysz siatkę przycisków modułów (np. Infoczytnik, Skarbiec Danych, Kalkulator, Rzut kośćmi).
3. Kliknięcie przycisku otwiera wybrany moduł.

### Tryb użytkownika i tryb administratora
1. Domyślnie działa widok użytkownika.
2. Aby włączyć widok administratora, dopisz w adresie `?admin=1`.
3. W trybie administratora pojawiają się dodatkowe przyciski (np. Generator Nazw, Generator NPC, Audio) i informacje pomocnicze.

### Powiadomienia
1. Kliknij czerwony przycisk **Włącz powiadomienia** pod siatką modułów.
2. Przeglądarka zapyta o zgodę.
3. Po zaakceptowaniu urządzenie może odbierać komunikaty z modułu Infoczytnik.

### Gdzie są linki do mapy i obrazków
- Linki do przycisków **Mapa** i **Obrazki** są trzymane w pliku `Main/ZmienneHiperlacza.md`.
- Jeśli link ma się zmienić, podmień tylko adres URL po dwukropku.

## 🇬🇧 User instructions (EN)
Main is the launcher screen used to open all other modules.

### How to open it and what you will see
1. Open `Main/index.html`.
2. You will see a grid of module buttons (for example DataSlate, DataVault, Calculator, Dice Roller).
3. Clicking a button opens the selected module.

### User mode and admin mode
1. User mode is the default.
2. To enable admin mode, append `?admin=1` to the URL.
3. In admin mode, extra buttons appear (for example Name Generator, NPC Generator, Audio) together with helper notes.

### Notifications
1. Click the red **Enable notifications** button below the modules grid.
2. Your browser will ask for permission.
3. After approval, your device can receive DataSlate notifications.

### Where map and images links are stored
- **Map** and **Images** button URLs are stored in `Main/ZmienneHiperlacza.md`.
- To update them, replace only the URL after the colon.
