ADMINISTRATUM DATA VAULT
Kliknięcie przycisku wygeneruje nowe pliki data.json oraz firebase-import.json. Wskaż lokalny plik Repozytorium.xlsx z dysku, a następnie do Firebase Realtime Database importuj firebase-import.json.
Część danych jest domyślnie ukryta.
▸FILTRY
▸ Kliknij nagłówek, aby sortować (Shift = sort wielokolumnowy).
▸ Drugi wiersz nagłówka: filtry per kolumna (fragment / liczby).
▸ Zaznacz 2+ wiersze, aby porównać.
Jeśli tooltipy cech nie wyskakują na telefonie: stuknij w tag cechy.
Brak danych
