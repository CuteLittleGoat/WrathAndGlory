# Generator NPC

## 🇵🇱 Instrukcja dla użytkownika (PL)
Generator NPC pozwala zbudować kartę przeciwnika/NPC na podstawie danych z DataVault.

1. Otwórz `GeneratorNPC/index.html`.
2. Wybierz język (PL/EN).
3. Wybierz jednostkę z listy bestiariusza.
4. Dodaj opcjonalne notatki.
5. W sekcjach wyposażenia wybierz elementy (broń, pancerz, augmentacje, ekwipunek, talenty, psionika, modlitwy).
6. Kliknij **Generuj kartę**, aby zaktualizować podsumowanie.
7. Jeśli chcesz zacząć od zera, kliknij **Reset**.
8. Możesz zapisać konfigurację do „Ulubionych” i później ją odtworzyć.

## 🇬🇧 User instructions (EN)
NPC Generator lets you build an enemy/NPC card using DataVault content.

1. Open `GeneratorNPC/index.html`.
2. Choose language (PL/EN).
3. Select a bestiary unit.
4. Add optional notes.
5. In equipment sections, pick entries (weapon, armor, augmentations, gear, talents, psionics, prayers).
6. Click **Generate card** to update summary.
7. Use **Reset** to start over.
8. You can save setups in **Favorites** and reload them later.
